# Предсказание цен на квартиры

## Описание

Консольное приложение на Python обучает три регрессионные модели на данных о квартирах и сравнивает их по метрикам качества. Данные можно загрузить из CSV или сгенерировать синтетически с правдоподобной зависимостью цены в рублях от площади, комнат, этажа, удалённости от центра, года постройки и состояния. В проекте строятся графики сравнения предсказаний с фактом, остатков, корреляций и кривая обучения для лучшей модели. После обучения доступен интерактивный ввод параметров одной квартиры и вывод предсказанной цены в ₽.

## Установка и запуск

```bash

cd apartment_price_predictor
pip install -r requirements.txt
python main.py
```

После таблицы метрик программа запросит параметры квартиры; введите числа в указанных диапазонах.

## Как работает

1. Загружается `data/apartments.csv` или генерируется синтетический датасет с целевой переменной `price_rub`.
2. Данные проверяются на полноту колонок и пропуски, признаки и цена разделяются на обучающую (80 %) и тестовую (20 %) выборки.
3. Обучаются три модели: `Pipeline(StandardScaler, LinearRegression)`, `RandomForestRegressor`, `GradientBoostingRegressor`.
4. На тесте считаются MAE, RMSE, R² и MAPE; результаты выводятся таблицей (MAE/RMSE в формате с разделителями тысяч и символом ₽).
5. В интерактивном режиме вводятся признаки одной квартиры; цена предсказывается лучшей по R² моделью.
6. Сохраняются графики в каталог `plots/`, включая кривую обучения лучшей модели с `cross_val_score`.
7. В консоль выводится название лучшей модели и её R² на тесте.

## Модели и метрики

Ниже **приблизительные** значения для синтетических данных (фактические цифры зависят от выборки и шума):

| Модель                  | MAE (пример) | RMSE (пример) | R² (пример) | MAPE (пример) |
|-------------------------|-------------:|--------------:|------------:|--------------:|
| LinearRegression        | ~1,5–4 млн ₽ | ~2–5 млн ₽   | ~0,85–0,98  | ~5–15 %       |
| RandomForest            | ~2–5 млн ₽   | ~3–6 млн ₽   | ~0,80–0,95  | ~6–18 %       |
| GradientBoosting        | ~2–5 млн ₽   | ~3–6 млн ₽   | ~0,82–0,96  | ~6–17 %       |

*Значения ориентировочные; после смены формулы генерации и валюты на ₽ они могут отличаться.*

## Визуализация

1. **plot_1** — рассеяние «факт vs предсказание» для трёх моделей.
2. **plot_2** — столбчатые MAE, RMSE, R² и MAPE на тесте.
3. **plot_3** — горизонтальная важность признаков (Random Forest).
4. **plot_4** — остатки по трём моделям.
5. **plot_5** — тепловая карта корреляций признаков и цены.
6. **plot_6** — распределение цен в датасете.
7. **plot_7** — кривая обучения лучшей модели: R² на обучении и при кросс-валидации по размеру выборки (10 %–100 %).
